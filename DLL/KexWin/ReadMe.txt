【はじめに】
	[KexWin.DLL]をダウンロードして頂き、ど〜もです．(^^;ゞ
	[KexWin.DLL]は、HSP上で、Windowsのウィンドウを参照・変更する
	　DLLです．

【動作環境】
	HotSoupProcessor Ver. 2.4g以降が動作する環境

【インストール】
	ダウンロードした[KexWin[???].lzh]を解凍ツールなどで
	　解凍して下さい．( [???] はバージョン )

	    以下のファイルが解凍されます :
		    KexWin.DLL		: KexWin.dll本体
		    KexWin.as		: 命令インクルード用ヘッダファイル
		    KexWin_***.as	: サンプルスクリプト
		    ReadMe.txt		: この説明ファイル　
		    KexWin.txt		: 各 機能 / 命令 説明ファイル
		    Kex_Samp.cur	: カーソルファイル
		    Kex_Samp.ani	: アニメーションカーソルファイル
		    Rgn.bmp		: ImageRgnサンプル用BMPファイル
		    Rgn_24.bmp		: 〃	〃	〃	〃

	解凍された、[KexWin.dll]をHSPと同じフォルダに
	　コピーしてください．

【使用方法】
	[KexWin.txt]をご覧ください．

【アンインストール】
	解凍されたファイルを全て削除してください．
