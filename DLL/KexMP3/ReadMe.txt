【はじめに】
	[KexMP3.DLL]をダウンロードして頂き、ど〜もです．(^^;ゞ
	[KexMP3.DLL]は、HSP上で、MP3ファイルを再生させる事が
	　出来るようにするDLLです．


【動作環境】
	HotSoupProcessor が動作する環境
	　(動作確認は、HSP 2.5βR 2 〜)


【インストール】
	ダウンロードした[KexMP3[???].lzh]を解凍ツールなどで
	　解凍して下さい．( [???] はバージョン )

	    以下のファイルが解凍されます :
		    KexMP3.DLL		: KexMP3.DLL本体
		    KexMP3.as		: 命令インクルード用ヘッダファイル
		    KexMP3_sample.as	: サンプルスクリプト (簡易MP3プレイヤー)
		    KexMP3_sample_NEW.as: サンプルスクリプト (出力オプションつき)
		    ReadMe.txt		: この説明ファイル
		    KexMP3.txt		: 各 機能 / 命令 説明ファイル

	解凍された、[KexMP3.DLL]をHSPと同じフォルダにコピーしてください．


【使用方法】
	[KexMP3.txt]をご覧ください．


【アンインストール】
	解凍されたファイルを全て削除してください．
